# مرجع CLI

التوثيق الكامل لجميع خيارات سطر الأوامر في باطن.

## الصيغة العامة

```bash
batin [الخيارات العامة] <الأمر> [خيارات الأمر]
```

## الخيارات العامة

| الخيار | المختصر | الوصف |
|--------|---------|-------|
| `--json` | | إخراج النتائج بصيغة JSON |
| `--csv` | | إخراج النتائج بصيغة CSV |
| `--verbose` | `-v` | تفعيل التسجيل المفصل |
| `--help` | `-h` | عرض معلومات المساعدة |
| `--version` | `-V` | عرض معلومات الإصدار |

## الأوامر

### `scan` - فحص الملفات أو المجلدات

تحليل الملفات للكشف عن النوع وتقييم التهديدات.

```bash
batin scan <المسار> [الخيارات]
```

#### المعاملات

| المعامل | الوصف |
|---------|-------|
| `المسار` | مسار الملف أو المجلد للفحص |

#### الخيارات

| الخيار | المختصر | الوصف | الافتراضي |
|--------|---------|-------|----------|
| `--recursive` | `-r` | فحص المجلدات تكرارياً | `false` |
| `--output` | `-o` | مسار ملف الإخراج للنتائج | stdout |
| `--exclude` | `-e` | استبعاد الملفات المطابقة لنمط glob | لا شيء |
| `--min-threat` | | الحد الأدنى لمستوى التهديد للعرض | الكل |
| `--hash` | | تضمين MD5/SHA-256 | `false` |

#### أمثلة

```bash
# فحص ملف أساسي
batin scan document.pdf

# فحص مجلد تكراري
batin scan /home/user/Downloads --recursive

# استبعاد الأنماط
batin scan /project -r --exclude "*.log" --exclude "target/*"

# عرض الملفات الخطيرة فقط
batin scan /uploads -r --min-threat dangerous

# إخراج JSON مع الهاشات
batin scan /evidence -r --json --hash --output report.json
```

### `watch` - المراقبة في الوقت الحقيقي

مراقبة مجلد للملفات الجديدة والمعدلة في الوقت الحقيقي.

```bash
batin watch <المسار> [الخيارات]
```

#### المعاملات

| المعامل | الوصف |
|---------|-------|
| `المسار` | مسار المجلد للمراقبة |

#### السلوك

- يراقب تكرارياً بشكل افتراضي
- يزيل الأحداث المكررة (نافذة 200 مللي ثانية)
- ينتظر استقرار الملف قبل الفحص (50 مللي ثانية)
- يعرض النتائج مباشرة عند إنشاء/تعديل الملفات

#### أمثلة

```bash
# مراقبة مجلد التنزيلات
batin watch ~/Downloads

# المراقبة مع التسجيل المفصل
batin watch /var/log --verbose
```

## رموز الخروج

| الرمز | المعنى |
|------|--------|
| `0` | نجاح - تمت معالجة جميع الملفات |
| `1` | خطأ - الملف غير موجود، رفض الإذن، إلخ |

---

## متغيرات البيئة

| المتغير | الوصف | الافتراضي |
|---------|-------|----------|
| `BATIN_MAX_READ_BYTES` | الحد الأقصى للبايتات لقراءتها لكل ملف | 3072 |
| `BATIN_TIMEOUT_MS` | مهلة قراءة الملف بالمللي ثانية | 5000 |
| `NO_COLOR` | تعطيل الإخراج الملون | غير محدد |

---

:::tip نصيحة احترافية
اجمع بين إخراج `--json` و `jq` للتصفية القوية:

```bash
batin scan /uploads -r --json | jq '.[] | select(.file_type.threat_level != "Safe")'
```

:::
