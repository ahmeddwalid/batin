# مقدمة لباطن

**باطن** (من العربية "الخفي" أو "الداخلي") هي مكتبة احترافية مؤمنة للكشف عن أنواع الملفات مكتوبة بلغة Rust. على عكس الأدوات البسيطة التي تتحقق فقط من امتدادات الملفات، يفحص باطن المحتوى الفعلي للملفات للكشف عن طبيعتها الحقيقية.

## لماذا باطن؟

أدوات الكشف التقليدية لها قيود كبيرة:

| الأداة | القيد |
|------|------------|
| امتدادات الملفات | يمكن إعادة تسميتها/تزويرها بسهولة |
| البايتات السحرية الأساسية | تتحقق فقط من البايتات الأولى |
| أمر `file` | لا يوجد تقييم للتهديدات، كشف محدود للملفات متعددة الصيغ |

**باطن يذهب أبعد** من خلال الجمع بين تقنيات كشف متعددة:

```mermaid
flowchart LR
    A[إدخال الملف] --> B[البايتات السحرية]
    B --> C[تحليل الإنتروبيا]
    C --> D[كشف متعددي الصيغ]
    D --> E[فحص التهديدات]
    E --> F[نتيجة الكشف]
    
    style A fill:#1a1a2e
    style F fill:#25c2a0
```

## الميزات الرئيسية

### 🔍 كشف متعدد المراحل

باطن لا يفحص فقط البايتات السحرية. يحلل:

- **توقيعات البايتات السحرية** - مطابقة الأنماط مع أكثر من 60 صيغة
- **إنتروبيا شانون** - يكشف المحتوى المضغوط/المشفر
- **فحص متعدد الإزاحات** - يجد الصيغ المخفية (متعددي الصيغ)
- **فحص التهديدات المضمنة** - يجد الماكرو والسكربتات والملفات التنفيذية

### 🛡️ تصميم الأمان أولاً

- **صفر كود غير آمن** - ضمان سلامة الذاكرة
- **لا يتوقف** - مختبر ضد أي إدخال
- **قراءات محدودة** - يمنع هجمات استنفاد الذاكرة
- **حماية المهلة** - يمنع حجب الخدمة من الملفات المعيبة

### ⚡ أداء عالي

- **I/O غير متزامن** - عمليات ملفات غير محجوبة
- **معالجة متوازية** - حساب الإنتروبيا متعدد النوى
- **خوارزميات مرور واحد** - حسابات محسنة

### 🌍 متعدد المنصات

يعمل على Linux و Windows و macOS و FreeBSD.

## حالات الاستخدام

### تحليل البرمجيات الخبيثة

كشف الملفات التنفيذية المضغوطة، والملفات متعددة الصيغ (هجمات PDF+EXE)، والماكرو المضمن قبل تنفيذها.

### الطب الشرعي الرقمي

تحديد أجزاء الملفات، والتحقق من سلامة الملفات، وكشف تزوير الامتدادات أثناء التحقيقات.

### التدقيق الأمني

فحص المجلدات بحثاً عن ملفات مشبوهة، وتحديد انتهاكات السياسة، وإنشاء تقارير الامتثال.
